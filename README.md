# Audius-Bot
An all-in-one Discord bot for the @AudiusProject Discord server. Made to replicate the functionality of the `$AUDIO Bot` and `Audius Party Bot`, whilst adding QoL features like anti-scam Auto Moderation and more!

### Disclaimer
This project was purely made for fun; there is no expectation from me for this bot to be used in the Audius Discord server. If you would like to do so, please contact me at [`toby@tobezdev.com`](mailto:toby@tobezdev.com) (click) from an `@audius.co` email address :D
